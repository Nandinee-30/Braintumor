# 🧠 Brain Tumor Segmentation AI

An AI-powered Brain Tumor Detection and Segmentation system that uses Deep Learning and Medical MRI images to identify and analyze brain tumor regions.

This project aims to assist in medical image analysis by providing automated tumor detection and segmentation using Artificial Intelligence techniques.

## 🚀 Features

- Brain MRI image processing
- Brain tumor detection using Deep Learning
- Tumor region segmentation
- U-Net based segmentation model
- AI-based prediction system
- MRI scan analysis
- Automatic PDF report generation
- User-friendly web interface

## ⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/Brain_Tumor_Segmentation.git

Move into project directory:

cd Brain_Tumor_Segmentation

Create virtual environment:

python -m venv venv

Activate virtual environment:

venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

## ▶️ Run Application

Start Flask application:

python app.py

Open browser:

http://127.0.0.1:5000/

## 🧠 Working Process

1. User uploads Brain MRI image
2. Image preprocessing is performed
3. Deep Learning model analyzes MRI scan
4. Tumor region is detected and segmented
5. Prediction result is displayed
6. PDF report is generated

## 📊 Dataset

This project uses:

BraTS (Brain Tumor Segmentation Dataset)

The dataset contains MRI brain scan images used for training and testing the tumor segmentation model.
